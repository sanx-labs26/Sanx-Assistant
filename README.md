# Sanx-Assistant
An AI voice assistant built with Python and Livekit.

Features

- Real-time voice interaction
- AI-powered conversations
- Custom assistant personality
- Extensible architecture
- Future support for memory and preferences

Technologies

- Python
- LiveKit
- Gemini
- SQLite (planned)

Installation

git clone <repository-url>
cd Sanx-Assistant
pip install -r requirements.txt

Run

python main.py

Roadmap

- SQLite memory
- User preferences
- Error handling
- AI Study Buddy integration
