# Sanx-Assistant
Sanx Assistant is an AI-powered voice assistant built with Python and LiveKit. It provides real-time voice interaction, intelligent conversations, and a customizable assistant experience.

Features

- Real-time voice interaction
- AI-powered conversations
- Custom assistant personality
- Modular and extensible architecture
- Support for future memory and preference systems

Technologies Used

- Python
- LiveKit
- Gemini
- SQLite (Planned)

Installation

git clone <repository-url>
cd Sanx-Assistant
pip install -r requirements.txt

Usage

python main.py
